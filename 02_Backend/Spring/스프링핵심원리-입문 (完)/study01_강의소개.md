## :pushpin: 강의 소개

### 스프링 공부는 이렇게 해야 합니다.

### 간단한 웹 애플리케이션 개발해보기
- 스프링 프로젝트 생성
- 스프링 부트로 웹 서버 실행
- 회원 도메인 개발
- 웹 MVC 개발
- DB 연동 - JDBC, JPA, 스프링 데이터 JPA
- 테스트 케이스 작성
