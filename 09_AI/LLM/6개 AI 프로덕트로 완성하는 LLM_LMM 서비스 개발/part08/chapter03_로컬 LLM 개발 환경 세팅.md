- https://ai.meta.com/blog/meta-llama-3

### 로컬 LLM 개발 환경 세팅
로컬 LLM 활용에 필요한 기능들
1. 프롬프트 설계 도구
2. 벡터 기반 데이터베이스
3. 텍스트 토큰화 도구
4. GUI 개발 도구
5. 유사도 기반 검색 기능
6. 로깅 / 모니터링 /테스팅 도구
7. 메모리 관리 도구
8. 기타 텍스트 처리 파이프라인
9. 테스트 자동화 도구
10. 성능 최적화 및 튜닝 도구

Chroma
고속 벡터 데이터베이스. 
대용량 벡터 데이터를 관리하고, 임베딩 기반 유사성 검색을 통해 빠른 정보 검색을 지원하는 오픈소스 벡터 데이터베이스

langchain
생성형 AI 개발 툴킷
다양한 데이터베이스, 파일, API 등 다양한 소스와 연동
LLM을 이용한 멀티태스크 워크플로우 생성 가능
모델 연동, 프롬프트 파이프라인, 문서열람, 텍스트 처리, 파서 등

promptflow
프롬프트 설계와 관리 도구
LLM을 위한 프롬프트 설계, 테스트, 관리 작업을 효율적으로 지원하는 GUI 기반 도구
프롬프트 테스팅, 버전 관리, 워크플로우 편집, 실험 . 성능 추적 등

streamlit
데이터 앱/대시보드 제작 도구
빠른 프로토타이핑을 위한 개발 프레임워크
데이터 시각화 및 상호작용 요소, 데이터 자동 갱신 기능, 배포 및 공유 기능
