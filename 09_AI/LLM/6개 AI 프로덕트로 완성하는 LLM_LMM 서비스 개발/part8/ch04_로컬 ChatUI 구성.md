### ChatUI 구성 프레임워크 소개 및 비교
tkinter
- python 기본으로 제공되는 GUI 프레임워크
- 가벼운 애플리케이션을 개발하는데 유용
- 간단한 데스크탑 애플리케이션, 실험적 GUI 애플리케이션에 적합

streamlit
- 다양한 데이터 시각화 기타 화면구성 위젯 지원
- 간단한 코드로 UI 구성 가능, 대화형 UI 구현도 어렵지 않음
- 간단한 데스크탑 애플리케이션, 실험적 GUI 애플리케이션에 적합

gradio
- AI 모델 데모용 인터랙티브 앱
